<h1>Music Shop</h1>
<hr>
<p>This is an app that runs in terminal simulating a musical shop online. Users can:</p>
<ul>
  <li>Buy an instument</li>
  <li>Check inventory</li>
  <li>Restock the Inventory</li>
</ul>
<br>
<br>

<h1>Tech</h1>
<hr>
<ul>This app uses</ul>
  <li>SQL Database</li>
  <li>Node.js</li>
  <li>MySQL Node Package</li>
<br>
<br>
